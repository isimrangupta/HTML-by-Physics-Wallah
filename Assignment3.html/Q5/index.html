HTML
The standard markup language used to create the structure of web pages.

CSS
A stylesheet language used to describe the presentation of a web page, including layout, colors, and fonts.

JavaScript
A programming language used to create interactive effects and dynamic content on web pages.

React
A JavaScript library for building user interfaces, primarily for single-page applications.

Node.js
A runtime environment that allows developers to run JavaScript on the server-side to build scalable network applications.

MongoDB
A NoSQL database that stores data in JSON-like documents, known for its flexibility and scalability.

Git
A version control system used to track changes in source code during software development.
