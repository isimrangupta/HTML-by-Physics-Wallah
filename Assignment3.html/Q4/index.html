1. <html>: The root element that encompasses all other elements on the web page.
  2. <head>: Contains metadata, such as the title of the document, links to stylesheets, and scripts.
  3. <title>: Sets the title of the web page, which appears in the browser tab.
  4. <body>: The main content area of the web page where all visible elements are placed.
  5. <h1> to <h6>: Header tags used to define headings, with <h1> being the highest (most important) level and <h6> the lowest.
  6. <p>: Defines a paragraph of text.
  7. <a>: Creates a hyperlink to another page or resource, using the href attribute to define the link target.
  8. <img>: Embeds an image into the web page, with the src attribute specifying the image source.
  9. <ul>: Defines an unordered (bulleted) list.
  10. <ol>: Defines an ordered (numbered) list.
  11. <li>: Represents an individual list item within an unordered or ordered list.
  12. <div>: A block-level container used to group elements for styling or scripting purposes.
  13. <span>: An inline container used to group text or elements for styling purposes.
  14. <form>: Defines a form used to collect user input.
  15. <input>: An input field within a form, used for various types of user input, such as text, radio buttons, or checkboxes.
  16. <button>: Creates a clickable button.
  17. <table>: Defines a table.
  18. <tr>: Represents a table row.
  19. <td>: Represents a table cell.
  20. <th>: Defines a header cell in a table.
